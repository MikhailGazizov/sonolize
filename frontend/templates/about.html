About Sonolize

Sonolize is a tool that aggregates Audio and Image processing together to facilitate glitch-art creation with databending.

What is Databending?

Databending is an action of manual interference of raw data by deleting, adding and editing strings of bytes in media files in order to create visual and/or sound distortions.

Why Sonolize?

While databending is almost purely manual process, doing it without software might damage vital bytes of containers which will lead to corruption of file till extent of being not able to be open.

This is where Sonolize comes in. It parses the image and automates application of Digital Signal Processing preserving container bits from being redacted.

How does it work?

Computers store data in binary which means that both images and audio on your computer are streams of 1s and 0s, but the difference is how computer treats this data; Image can be opened as Audio and vice versa. In this particular instance Sonolize interprets image bytes as Pulse Code Modulation (PCM, is a method of coding audio signal) and then applies Digital Signal Processing used for music; like Delay and Compression. Afterward, processed audio is being translated back to image format
